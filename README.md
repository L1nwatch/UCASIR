# lucene_graduation_design

毕设工程实现，主要是基于 Lucene 平台研究 HITS 算法

## 环境说明

* macOSX 10.10.5
* IDEA 2016.2
* JDK 1.8
* Lucene-4.3

实现了 HITS 算法中有关收敛算法部分的关系链接库, HITS 算法起作用了, 原来是排序出了点问题, 修正了

### 2017.05.05

由于链接关系数据库内容的改变, 顺带修改了 SQL 交互的调试代码
